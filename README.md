# 🌍 TravelPal - 智能旅行规划App

> "一个有灵魂的旅行App，记住你的每一次冒险"

## 📋 功能规划

### 核心功能
1. **创建旅行计划**
   - 旅行名称、日期、目的地
   - 行程安排（天数、景点、交通）
   - 预算规划

2. **邀请协作**
   - 邀请朋友/伴侣一起编辑
   - 实时同步
   - 权限管理（管理员/编辑者/查看者）

3. **智能规划**
   - AI推荐景点、餐厅
   - 基于预算推荐
   - 时间线优化

4. **清单管理**
   - 行李清单
   - 待办事项
   - 购物清单

5. **地图视图**
   - 景点地图
   - 路线规划
   - 附近设施

---

## 🏗️ 技术架构

### 前端
- React + TypeScript
- TailwindCSS（样式）
- React Router（路由）
- Zustand（状态管理）

### 后端/数据
- **Supabase**（开源Firebase替代）
  - PostgreSQL数据库
  - 实时订阅
  - 用户认证
  - Row Level Security

### 实时协作
- Supabase Realtime
- WebSocket同步

### 特色功能
- AI推荐引擎（基于用户偏好）
- 记忆系统集成（记住用户喜好）
